# multiple-services-web

- A frontend implementation of multiple gRPC services.

## Development

### Commands

```console
# run dev server
npm run dev

# build
npm run build

# deploy to Cloudflare Pages
npm run deploy
```

### Set API URL

create `.env.local` and set `VITE_API_BASE_URL` value.

```
VITE_API_BASE_URL=xxx
```
